---

| Category | Tool | What It's Best At | When to Choose It |
|---|---|---|---|
| **Product Analytics** | **Amplitude** | Events, funnels, retention, cohorts | Best overall default for mobile apps |
|  | **Mixpanel** | Funnels + retention, simpler UI | Strong alternative to Amplitude |
|  | **PostHog** | Events + feature flags + self-host | Budget or data-ownership focused teams |
| **Crash & Performance** | **Sentry** | Crashes, performance tracing, breadcrumbs | Best-in-class debugging |
| **Logging (Server & App)** | **Datadog** | Logs, metrics, APM | Scales well, expensive but powerful |
|  | **AWS CloudWatch** | Logs + metrics | Best if AWS-native |
|  | **ELK Stack** | Full control logging | Infra-heavy teams |
| **All-in-One / Engagement** | **Braze** | Analytics + messaging | Mature lifecycle & CRM motion |
| **Warehouse / Raw Events** | **BigQuery** | Event storage & analysis | GCP stacks |
|  | **Snowflake** | Analytics at scale | Enterprise data teams |
|  | **Redshift** | AWS-native warehousing | AWS-first teams |
