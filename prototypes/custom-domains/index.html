GoDaddy — how to add records

  1. 1 Log in to GoDaddy and open DNS Management for your domain.
  2. 2 Add an A record — set Name to @, Value to 76.76.21.21, TTL to 1 hour.
  3. 3 Add a CNAME record — set Name to www, Value to cname.vercel-dns.com, TTL to 1 hour.
  4. 4 Save your changes. DNS can take a few minutes to several hours to propagate.
